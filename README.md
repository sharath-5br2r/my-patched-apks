# Eden CI Workflow Fork 

This is a fork of [Eden-CI/Workflow](https://github.com/Eden-CI/Workflow) optimised for integration with my-patched-apks
Currently only Android optimised and Android Chromeos versions of Eden Emulator is built.
Forgejo releases are removed and replaced with this repo.

# Orginal Readme

This is the master repository containing the entire CI and build suite for Eden. This repository serves to dispatch successful builds to individual repositories for the master branch, pull requests, and releases. Nightlies will come at a later date.
